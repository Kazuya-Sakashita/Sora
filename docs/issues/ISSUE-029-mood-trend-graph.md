# ISSUE-029: 気持ちトレンドグラフ

name: ISSUE-029
type: feature
priority: Medium
status: open

---

## 概要

feelings の蓄積データを折れ線・棒グラフで可視化し、感情の変化を振り返れるようにする。

## 背景

気持ち記録が蓄積しても現在は一覧表示のみで「傾向」がわからない。グラフ化により記録の意味が増し、継続モチベーションになる。ロス後ユーザーには「回復の可視化」になる。

## 目的

感情記録の蓄積を「成長・変化の証明」として見せ、記録継続率を上げる。

## スコープ

- feelings 画面に「トレンド」タブを追加（既存の記録一覧タブと切替）
- 過去30日の気持ちタグ推移を棒グラフで表示（`recharts` 既インストール済み）
- タグ別フィルター（happy / calm / grateful など）
- 週次サマリー: 「今週は😌 おだやかが多かったです」のテキスト表示
- `/api/pets/{petId}/feelings` の既存APIを活用（新規エンドポイント不要）

## 非スコープ

- AI によるコメント・アドバイス
- 他ユーザーとの比較
- 月次・年次の長期グラフ（初回実装は30日のみ）

## 受け入れ条件

- [ ] 過去30日の気持ち推移が棒グラフで表示される
- [ ] タグ別フィルターが動作する
- [ ] 週次サマリーテキストが正しく生成される
- [ ] データが5件未満のときは「もっと記録するとトレンドが見えます」と表示
