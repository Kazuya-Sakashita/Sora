# ISSUE-079: チャットサブタイトル「いつでもそばにいるよ」をAI依存を助長しない表現に改める

name: ISSUE-079
type: bug
priority: Critical
status: done

---

## 背景

`chat.tsx` のヘッダーサブタイトルが「いつでもそばにいるよ」になっている。この表現は:

1. AIが「常にそばにいる存在」であることを強く示唆する
2. 人間の感情的サポートやプロフェッショナルなケアの代替になり得ると誤認させる
3. 特に急性悲嘆期のユーザーに「Soraがいれば大丈夫」という過度な依存を形成するリスクがある

Soraは「話せる場所」であって、「いつでもそばにいる存在」ではない。この1行がSoraの思想的根幹と矛盾している。

## 問題

```tsx
// chat.tsx
<p className="text-xs text-muted-foreground">いつでもそばにいるよ</p>
```

「いつでも」「そばにいる」という表現は、AIへの感情的依存を過度に肯定する。グリーフケアの文脈では、人間関係・専門家サポートへの回帰が重要であり、AIがその代替を担うことを示唆するコピーは倫理的に問題がある。

## 目的

チャット画面が「ここで話せる」という場所の提供者であることを伝えつつ、永続的な感情的依存先であるという誤認を排除する。

## スコープ

- `chat.tsx` のサブタイトルを変更:
  - `いつでもそばにいるよ` → `話せる場所があります`
- ヘッダータイトルはISSUE-075で別途対応（同時に実装可）

## やらないこと

- チャット機能・APIの変更
- ヘッダーUI構造の変更
- タイトル（ISSUE-075）との統合（別ISSUEで管理）

## 受け入れ条件

- [ ] チャットサブタイトルが「話せる場所があります」になっている
- [ ] AIが永続的な感情的依存先であるという誤認を生まない表現になっている
- [ ] `pnpm tsc --noEmit` 通過
- [ ] `pnpm test` 全通過
- [ ] 既存導線を壊していない

## 確認観点

- ユーザーの心を傷つけない
- Soraらしさを壊していない（「話せる場所」という控えめな表現）
- AI依存を強めすぎていない（最重要確認項目）
- 既存機能を壊していない
