# ISSUE-097: 3件記録達成後に「蓄積実感」メッセージを表示する

name: ISSUE-097
type: feature
priority: Medium
status: done

---

## 概要

3件目の記録を完了したタイミングで、記録の蓄積価値を伝える特別メッセージを1回だけ表示し、継続動機を植える。

## 背景

Hook Modelの「Variable Reward」と「Investment」の接続点として、3件という早い段階でユーザーに"蓄積の価値"を言語化して伝える必要がある。

"Aha Moment"の仮説は「3枚目の記録で初めて蓄積している感覚が生まれる」。ここで価値を明示することで、4件目以降の習慣化率が上がると期待する。

## 目的

3件達成という感情が高まるタイミングに介入し、「続けることの意味」をユーザーに納得させる。

## スコープ

- 3件目の記録保存完了後、AIリアクションの後続としてGlassCardを表示
- メッセージ例: 「3つの記録が積み重なりました。これが{pet.name}との時間の貯金です。1年後、きっと宝物になっています。」
- localStorage で表示済みフラグを保存（1回のみ表示）
- 4件目以降は表示しない

## 非スコープ

- 10件・50件などの追加マイルストーン（ISSUE-083で30件は実装済み）
- アニメーション演出の複雑化

## API変更

なし

## 受け入れ条件

- [ ] 3件目の記録後にメッセージが表示される
- [ ] ペット名がメッセージに入っている
- [ ] 2件目・4件目には表示されない
- [ ] 再起動後も再表示されない

## テスト観点

- 正常系: 3件目でメッセージ表示
- 境界値: 2件では非表示、4件目以降も非表示
- 永続化: localStorage保存後は再表示しない
