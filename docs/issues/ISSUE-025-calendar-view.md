# ISSUE-025: 記録カレンダービュー

name: ISSUE-025
type: feature
priority: Critical
status: done

---

## 概要

タイムライン画面にカレンダービューを追加し、「記録した日」を視覚化して蓄積感を生む。

## 背景

現在のタイムラインはスクロールリスト。記録が増えても「どれだけ残してきたか」が一目でわからない。カレンダーで記録の密度を見せることで「空白を埋めたい」動機が生まれる。

## 目的

記録の蓄積を「面」として可視化し、継続率と1日あたり記録数を上げる。

## スコープ

- タイムライン画面上部にカレンダーセクションを追加（月単位、スワイプで月移動）
- 記録のある日に一次色のドット、写真付き記録はサムネイルドット表示
- カレンダーの日付タップ → その日の記録にスクロール
- 現在はリストビューとカレンダービューをトグルで切り替え
- `react-day-picker`（既インストール済み）を活用

## 非スコープ

- 感情カレンダー（ヒートマップ）← ISSUE-029 で分離
- カレンダーへの直接記録追加

## 受け入れ条件

- [ ] 記録のある日にドットが表示される
- [ ] 日付タップで該当記録にジャンプする
- [ ] 月移動ができる
- [ ] リスト/カレンダーのビュー切替ができる
