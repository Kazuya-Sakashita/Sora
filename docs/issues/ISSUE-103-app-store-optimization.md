# ISSUE-103: App Store / Google Playのストア掲載を最適化する

name: ISSUE-103
type: docs
priority: High
status: done

---

## 概要

App Store・Google Playのスクリーンショット・説明文・キーワードを「生前から使う記録アプリ」の訴求に最適化し、オーガニック流入を増やす。

## 背景

バズりの起点としてSNSシェア（Loop 1）が機能し始めると「これ何のアプリ？」からApp Storeに来るユーザーが増える。その際にストアページが「ペットロスアプリ」に見えると、生前ユーザーがダウンロードしない。

また、App Storeの検索キーワードを最適化することで、「犬 記録 アプリ」「ペット 思い出」などのオーガニック検索からの流入も見込める。

## 目的

ストアページをSNS流入の受け皿として最適化し、インストール転換率（CVR）を上げる。

## スコープ

- **スクリーンショット改善**（5〜8枚）:
  1. 「うちの子との●日目記念カード」ビジュアル（最も感情に刺さるシーンをトップに）
  2. タイムラインの記録一覧（写真が並ぶ様子）
  3. 気持ちの記録画面
  4. マイルストーンカード
  5. 「もしものときも、記録が支えになる」グリーフケアの価値訴求
- **説明文改善**:
  - キャッチ: 「うちの子との毎日を、未来への贈り物に」
  - 生前から使うことの価値を前面に出す
  - グリーフケア機能は2段落目以降に
- **キーワード設定**（iOS）:
  - 犬, 猫, ペット, 記録, 思い出, 日記, アルバム, グリーフ, ロス, 記念日
- 制作物: スクリーンショット用デザインファイル（Figma）+ 説明文テキスト
