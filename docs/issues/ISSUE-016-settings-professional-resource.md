# ISSUE-016: 設定画面に専門家リソースへのリンクを追加する

## 基本情報

| 項目 | 内容 |
|------|------|
| 優先度 | medium |
| ステータス | open |
| 関連画面 | 設定画面 |
| 関連ファイル | `components/screens/settings.tsx`（または設定コンポーネント） |

## 問題の概要

倫理設計として定義された「医療・カウンセリングとの線引き」が未実装。深刻な悲しみを抱えるユーザーに向けた専門家リソースへの案内が設定画面にない。

## 詳細

仕様再考（`docs/specs/sora-product-rethinking.md`）の倫理設計セクションで、以下が定義された:

> 「このアプリは、ペットとの思い出を大切にするためのものです。深刻な悲しみや精神的な不調を感じている場合は、専門家のサポートをご検討ください。」

この文言と専門家リソースへのリンクを設定画面に静かに配置する。

**設計の原則:**
- 目立たせない（設定の下部に小さく）
- 押しつけない（誰でも気づかなくてよい程度）
- ただし必要な人には確実に届く場所にある

## 期待する動作

設定画面の最下部に、以下のセクションを追加:

```
---
サポートについて

このアプリはペットとの思い出を大切にするためのものです。
深刻な悲しみや日常生活への影響を感じている場合は、
専門家のサポートもご検討ください。

[心の健康について（外部リンク）]
```

リンク先: 厚生労働省「こころの健康相談統一ダイヤル」または同等のリソース

## 実装方針

1. 設定画面の最下部に「サポートについて」セクションを追加
2. テキストリンクまたは小さいボタンで外部URLを開く（`target="_blank"`）
3. フォントサイズは小さめ（`text-sm`）、色は控えめ（`text-muted-foreground`）

## 完了条件

- [ ] 設定画面の下部に「サポートについて」テキストが表示される
- [ ] リンクをタップすると外部ページが開く
- [ ] 目立ちすぎない・押しつけがましくない表示になっている
- [ ] 既存機能に影響なし確認済み
- [ ] ステータスを done に更新した
