# ISSUE-010: チャットのクイックプロンプトがロス後ユーザー専用

## 基本情報

| 項目 | 内容 |
|------|------|
| 優先度 | medium |
| ステータス | open |
| 関連画面 | お話しする（チャット）画面 |
| 関連ファイル | `components/screens/chat.tsx` |

## 問題の概要

チャット画面のクイックプロンプトが「会いたいな」「今日は少しつらい」などロス後向けのみで、生きているペットとの日常を語るユーザーには不自然。

## 詳細

```tsx
// 現状（問題）
const quickPrompts = [
  "会いたいな",        // ロス後
  "今日は少しつらい",  // ロス後
  "思い出を話したい",  // ロス後
  "ありがとうを伝えたい", // ロス後
]
```

また、チャット画面の初期メッセージ「ここにいるよ。無理しなくて大丈夫」もロス後の文脈。生前ユーザーが開くと世界観のズレを感じる。

## 期待する動作

生前・ロス後どちらにも使えるプロンプトに変更する。

```tsx
// 修正案
const quickPrompts = [
  "今日のことを話したい",
  "思い出を話したい",
  "今日は少し気持ちが重い",
  "うれしいことがあった",
]
```

初期メッセージも「こんにちは。今日どんな一日でしたか？」など生前でも自然な文言に変更。

## 完了条件

- [ ] クイックプロンプトが生前ユーザーにも自然に使える
- [ ] 初期メッセージが生前・ロス後どちらにも対応した文言になった
- [ ] 既存機能に影響なし確認済み
- [ ] ステータスを done に更新した
