# ISSUE-004: ホームのグリーティング文言が生前利用に違和感

| 項目 | 内容 |

## 問題の概要

現在のホーム画面の挨拶文「今日はどんな気持ちですか？」は、ペットが元気な日常利用には重い。

## 詳細

仕様書では「ホームを『ダッシュボード』にしない。『今日もいっしょに』の感覚」と定義している。「今日はどんな気持ちですか？」という問いかけは、ペットロスを経験したユーザーには適切だが、日常利用の生前フェーズでは感情負荷が高すぎる。  
また、「ゆっくりで大丈夫です」もロス後向けのコピーで、生前ユーザーには違和感がある。

## 期待する動作

- 時間帯に応じた自然な挨拶（「おはようございます」等）
- ペットの名前を使った温かいコピー
- 「ゆっくりで大丈夫です」は設定でロス後フェーズのユーザーのみ表示するか、削除

## 実装方針

- 時間帯別挨拶（朝/昼/夜）を実装
- ペット名を使った文言に変更（「[名前]との今日を残しませんか」等）
- 下部のGentleMessageカードはフェーズに応じて変更（Phase 2対応予定のためいったん削除も可）
