## 問題の概要

現在のホーム画面のグリーティング「今日はどんな気持ちですか？」はロス後ユーザー向けで、日常利用に違和感がある。MVP再定義に伴い、**「今日で〇〇日目」カウンターをホームのメインコンテンツとし、ホーム画面を全面再設計する。**

## 詳細

仕様再考（`docs/specs/sora-product-rethinking.md`）でホーム画面の設計方針が更新された。

**旧方針:** 「今日はどんな気持ちですか？」（感情を問いかける）  
**新方針:** 「今日でリクと一緒に〇〇日目」（蓄積の可視化・毎日変わる数字）

「今日で〇〇日目」は:
- 毎日数字が増える → 毎日開く理由になる
- ペットを迎えた初日から計算できる → 新規ユーザーにも価値がある
- ロス後も「一緒にいた日数」として永遠に残る → 生前・ロス後をシームレスにつなぐ

## 期待する動作

**ホーム画面の新構成:**

```
[ヘッダー] ペット写真アイコン + 時間帯別挨拶（おはようございます / こんにちは / こんばんは）

[メインカウンター]
  🐾 リクと一緒に
      847 日
  （大きく・中央・温かいフォント）

[クイック記録ボタン]
  「今日の思い出を残す」（大きなCTA）

[直近の記録カード] 最新2〜3件

[今日の予定] あれば表示（なければ非表示）
```

**削除するもの:**
- 「今日はどんな気持ちですか？」の問いかけ
- 「ゆっくりで大丈夫です」のGentleMessageカード

**日数計算ロジック:**
- ペットプロフィールの「迎えた日（brought_at）」から今日までの日数
- 誕生日ではなく「迎えた日」を基準にする

## 実装方針

1. `home.tsx` のグリーティングを時間帯別挨拶に変更
2. 「今日で〇〇日目」カウンターコンポーネントを追加
   - `brought_at` から `today` までの差分日数を計算
   - 数字を大きく表示（ISSUE-011と連携）
3. GentleMessageカードを削除
4. クイック記録ボタンを目立つCTAに変更
