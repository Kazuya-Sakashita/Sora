# ISSUE-098: 「うちの子●日目」月次シェアカードをInstagramストーリー対応にする

name: ISSUE-098
type: improvement
priority: Critical
status: open

---

## 概要

毎月自動生成される「うちの子●日目」カードをInstagramストーリーサイズ（9:16縦型）に対応し、1タップでシェアできるビジュアルクオリティに引き上げる。SNS拡散ループ（Growth Loop 1）の起点となる最重要施策。

## 背景

Soraのバズり方はユーザーが「うちの子かわいい」コンテンツをSNSに投稿することで新規ユーザーが発見するフロー（Content Loop）。現状のシェアカードはWeb Share APIで画像を共有しているが、Instagramストーリーへの直接投稿に最適化されていない。

縦型9:16のビジュアルがInstagramストーリー・TikTok・Reelsで自然に使われるフォーマット。このサイズへの最適化が拡散率を大きく左右する。

## 目的

ユーザーが「これ投稿したい」と感じるビジュアルクオリティのシェアカードを提供し、SNSからの自然流入を生み出す。

## スコープ

- 月次カード（毎月1日自動生成）を9:16縦型レイアウトで生成
- デザイン要素:
  - ペット写真（最新記録から自動選択）をメインビジュアル
  - 「{pet.name}と{N}日目」の大きなタイポグラフィ
  - 今月の記録件数・気持ちサマリー
  - Soraのロゴ（控えめに）
- canvas APIで画像生成 → PNG ダウンロード + Web Share API
- ホーム画面のカードからシェアボタンを1タップで呼び出せる
- 横型（16:9）も引き続きサポート（切り替えボタン）

## 非スコープ

- 動画（mp4）出力（ISSUE-099で別途対応）
- ストーリーへの直接API投稿（Instagram Graph APIは個人ユーザー不可）
- カスタムテーマの選択UI
