# ISSUE-022: memories/feelingsの取得をページネーション化する

## 基本情報

| 項目 | 内容 |
|------|------|
| 優先度 | medium |
| ステータス | open |
| 関連画面 | タイムライン画面 |
| 関連ファイル | `lib/app-context.tsx`, `components/screens/timeline.tsx` |

## 問題の概要

初期化時に全記録を一括取得している。365日記録ユーザーで初期ロード1〜3秒の遅延が発生し得る。

## 期待する動作

- 初期取得は最新20件のみ
- タイムラインを下までスクロールで追加20件を読み込む（「もっと見る」ボタンも可）
- 記録数が多いユーザーでもホーム初期ロードが高速

## 実装方針

1. `app-context.tsx` の初期fetchを `?limit=20&offset=0` に変更
2. `app-context.tsx` に `loadMoreMemories()` 関数を追加
3. `timeline.tsx` に「もっと見る」ボタンを追加（無限スクロールは Phase 2）

## 完了条件

- [ ] 初期取得が20件に制限される
- [ ] 「もっと見る」で追加取得できる
- [ ] 全件取得済みなら「もっと見る」が非表示
- [ ] ステータスを done に更新した
