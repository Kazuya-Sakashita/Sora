# ISSUE-019: ホームに「今日の記録状態」を反映する

## 基本情報

| 項目 | 内容 |
|------|------|
| 優先度 | high |
| ステータス | done |
| 関連画面 | ホーム画面 |
| 関連ファイル | `components/screens/home.tsx` |

## 問題の概要

今日記録済みでも未記録でもホーム画面が同じ表示。「今日やった」という達成感が残らず、翌日の記録動機が生まれない。

## 期待する動作

- 今日未記録：現状通り「今日の思い出を残す」CTA
- 今日記録済み：「今日も残せました ✓」＋最新記録のサムネイル表示
- 写真あり記録ならサムネイルを小さく表示

## 実装方針

```ts
const today = new Date().toISOString().split("T")[0]
const recordedToday = memories.find((m) => m.date === today)
```

- `recordedToday` があればCTAを達成状態に切り替え
- タップでタイムラインへ遷移
