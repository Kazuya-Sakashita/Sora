# ISSUE-102: 記録の蓄積量をホーム・設定画面で可視化する

name: ISSUE-102
type: improvement
priority: Medium
status: open

---

## 概要

ホームまたは設定画面に「あなたは●日間、●件の思い出を記録しました」のスタッツを表示し、蓄積の重みをユーザーに実感させる。

## 背景

Hook Modelの「Investment」フェーズは、蓄積量が増えるほどスイッチングコストが上がりリテンションが向上する。しかし現状はユーザーが自分の蓄積量を一目で把握できる画面がない。

「1,200日分の記録がここにある」という感覚は、アプリを手放せなくなる最大の理由になる。

## 目的

ユーザーが蓄積の重みを実感することで、継続動機を強化しチャーンを防ぐ。

## スコープ

- ホーム画面のプロフィールセクションに以下を追加:
  - 「{pet.name}との記録：●件」
  - 「お迎えから：●日目」（既存のカウンターを流用）
  - 「記録期間：●ヶ月」
- 設定画面のペット情報セクションにも同じスタッツを表示
- 数値はメモリーAPIの件数とpet.broughtAtから計算（既存データで完結）

## 非スコープ

- 詳細な分析ダッシュボード
- 写真枚数・動画数のカウント

## API変更

なし（既存データから計算）

## 受け入れ条件

- [ ] ホーム画面で記録件数が表示される
- [ ] お迎えからの日数と一緒に表示されて文脈が伝わる
- [ ] 記録が0件でも壊れない（「まだ記録がありません」）

## テスト観点

- 正常系: 記録あり・記録なし
- 境界値: broughtAtがnullの場合
