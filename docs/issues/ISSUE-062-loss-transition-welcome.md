# ISSUE-062: ロスケア移行後72時間の「そばにいます」カードを表示する

name: ISSUE-062
type: improvement
priority: Critical
status: done

---

## 背景

虹の橋を渡った直後の72時間は、ユーザーにとって最も孤独で危機的な時間帯。現状のアプリはステータスが変わるだけで、移行直後に特別な接触がない。「このアプリはわかってくれている」という体験が最も必要な瞬間に最も薄い。

## 問題

`updatePetStatus("rainbow_bridge")` 後、ホーム画面は「過ごした〇〇日」に変わるだけ。移行直後と1ヶ月後で同じUIが表示される。最初の72時間のユーザーの心理状態への配慮がない。

## 目的

rainbow_bridge への移行から72時間以内のホーム画面に、1枚だけ寄り添いカードを表示する。「急がなくていい。ゆっくりでいい」というメッセージで安心感を届ける。

## スコープ

- rainbow_bridge ステータスになった時点を `sora:loss-transition-{petId}` に localStorage 記録
- ホーム画面で 72 時間以内かどうかを判定し、特別カードを表示
- カードには「ことば」「はなす」への控えめな誘導リンクを1つ
- ユーザーが手動で閉じることもできる（`sora:loss-welcome-dismissed-{petId}` フラグ）
- 72時間経過またはDismiss後は自動的に消える

## やらないこと

- プッシュ通知の追加
- 専門家へのルーティング（ISSUE-063で対応）
- 特別なアニメーション演出（シンプルなカードのみ）

## 受け入れ条件

- [ ] rainbow_bridge 移行直後（72時間以内）に寄り添いカードが表示される
- [ ] 72時間経過後はカードが表示されない
- [ ] 「閉じる」タップでDismissでき、その後は表示されない
- [ ] alive 状態ではカードが表示されない
- [ ] `pnpm tsc --noEmit` 通過
- [ ] `pnpm test` 全通過
- [ ] gstack で動作確認済み
