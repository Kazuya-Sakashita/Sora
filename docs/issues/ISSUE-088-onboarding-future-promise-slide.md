# ISSUE-088: オンボーディング最終画面に「記録の未来価値」スライドを追加する

name: ISSUE-088
type: improvement
priority: High
status: done

---

## 背景

現在のオンボーディングは初回記録完了後にAI価値訴求ステップ（ISSUE-073）が表示される。しかし「Soraがいつか自分を支えてくれる」という文脈の植え付けが不十分で、alive 期ユーザーが「なぜ記録し続けるのか」を腑に落とせないまま日常使いに入る。

```tsx
// first-record.tsx — AI価値訴求は「✨ 記録が、Soraとのつながりになります」のみ
// 未来の意味（いつか自分を支える）への接続がない
```

## 問題

初回記録完了後のAI価値訴求が「Soraを使いこなせそう」という機能訴求にとどまっており、「なぜ今記録するのか」という感情的な動機付けになっていない。

## 目的

初回体験の最後に「この記録は、いつかあなた自身を支えます」という未来への橋渡しを静かに届ける。alive 期のユーザーが「義務感」でなく「意味」で記録を続けられる土台を作る。

## スコープ

- `components/screens/first-record.tsx` のAIバリューステップを更新:
  - 現在の `showAiValueStep` ステップのテキストを以下に変更:
    - タイトル（絵文字なし）: 「記録は、いつかあなたを支えます」
    - 説明文: 「{pet.name}との時間を残すことは、今の喜びになります。そしてその積み重ねが、いつかSoraの言葉になります。急がなくていい。ただ、残していくだけでいい。」
    - ボタン: 「はじめる」（変更なし）
  - 絵文字は使用しない（重みのある言葉に絵文字は不釣り合い）

## やらないこと

- オンボーディングのステップ追加（現状の1ステップを更新するのみ）
- スライド形式への変更
- skip の追加
