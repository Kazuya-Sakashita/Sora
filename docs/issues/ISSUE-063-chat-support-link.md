# ISSUE-063: チャット画面に専門サポートへの控えめな導線を追加する

name: ISSUE-063
type: improvement
priority: Critical
status: done

---

## 背景

グリーフケアのAIチャットは、ユーザーが危機的状態にあるときに唯一の相談相手になりうる。しかし現状はチャット画面内に専門家への出口がなく、エスカレーション経路がゼロ。専門サポートへのリンクは Settings の最下部にのみ存在し、チャット中のユーザーが気づく導線がない。

## 問題

`chat.tsx` にはヘルプ・危機対応・専門機関への動線が一切ない。AIが対応できない深刻な状態でも会話を継続するしかない構造。安全設計として不十分。

## 目的

チャット画面のヘッダーに目立たない（しかし必要なときに見える）専門サポートへの導線を設置する。押しつけではなく、「ここにある」という安心感を提供する。

## スコープ

- チャットヘッダー右上に小さな `HeartHandshake` アイコンボタンを追加
- タップで小さなボトムシートを表示
  - 「つらくなったとき、専門家に話すことも選択肢のひとつです」
  - 厚生労働省リンク（settings.tsx の既存リンクと同じ）
  - 閉じるボタン
- 色は `text-muted-foreground/40`（通常時は目立たない）

## やらないこと

- 自動的に危機状態を検知する機能
- メッセージ内容に基づくアラート機能
- 強制的なポップアップ

## 受け入れ条件

- [ ] チャットヘッダー右上にアイコンボタンが表示される
- [ ] タップでサポート情報シートが開く
- [ ] 厚生労働省リンクが正しく機能する
- [ ] 閉じるで元のチャットに戻れる
- [ ] `pnpm tsc --noEmit` 通過
- [ ] `pnpm test` 全通過
- [ ] gstack で動作確認済み
