# ISSUE-024: 記録ストリーク（連続記録日数の可視化）

name: ISSUE-024
type: feature
priority: Critical
status: open

---

## 概要

連続記録日数（ストリーク）をホーム画面に表示し、毎日アプリを開く動機を作る。

## 背景

現状、記録しても「今日も残せました」トースト1回で完結する。翌日に戻る理由がない。ストリークは「やめたくない」心理を利用した最もシンプルな継続フック。

## 目的

D7継続率を現状推定15%以下から30%以上に引き上げる。

## スコープ

- `lib/streak.ts` に連続日数計算ロジックを実装（memories の date 配列から算出）
- ホーム画面のデイズカウンターカードにストリーク数を追加表示
  - 「🔥 3日連続記録中」のバッジUIを追加
- ストリーク途切れ時は「昨日の記録がまだです」をホームCTAに反映
- 7日・30日・100日の節目でホーム画面に祝福メッセージを表示

## 非スコープ

- ストリーク凍結（有料機能）
- プッシュ通知との連動（ISSUE-018 で既存）
- ランキング・他ユーザー比較

## 受け入れ条件

- [ ] 連続記録日数が正しく計算される（1日おきは連続にならない）
- [ ] ホームにストリーク数が表示される
- [ ] 7/30/100日の節目でお祝いメッセージが出る
- [ ] 記録がない日はストリークが0にリセットされる
