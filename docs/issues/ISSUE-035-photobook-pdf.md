# ISSUE-035: フォトブックPDF出力を実装する（Sora+）

name: ISSUE-035
type: feature
priority: High
status: open

---

## 概要

月単位の思い出を写真グリッド形式の PDF フォトブックとして出力できる機能を Sora+ 限定で実装する。

## 背景

記録のアウトプット体験がないと「お金を払う理由」が感情のみになり弱い。PDF フォトブックは「手元に残るもの」として有料プランの具体的な価値を示す。印刷所連携への布石にもなる。

## 目的

Sora+ の「形になるアウトプット」価値を提供し、月次継続課金の動機を強化する。

## スコープ

- `@react-pdf/renderer` で PDF テンプレート実装（写真グリッド2列・タイトル・日付・気持ちタグ）
- `GET /api/pets/:petId/photobook?year=YYYY&month=MM` — PDF 生成・返却（Plus 限定）
- タイムライン画面の月ヘッダーに「📖 この月のフォトブックを作る」ボタン追加
  - Plus ユーザー: クリックでダウンロード開始
  - Free ユーザー: Upgrade モーダル表示

## 非スコープ

- 印刷注文・物理配送
- テンプレート複数選択（初回は1種）
- 年間まとめフォトブック（ISSUE-032 の年次レポートと分離）

## 受け入れ条件

- [ ] PDF が月単位で生成・ダウンロードできる
- [ ] 写真なしの月はプレースホルダー（🐾）が表示される
- [ ] Free ユーザーはボタンを押すと Upgrade モーダルが出る
- [ ] Plus ユーザーは 402 を受け取らない
- [ ] 生成に10秒以上かかる場合はローディング表示
