# ISSUE-036: マイルストーン記念カードのPNG保存を実装する（Sora+）

name: ISSUE-036
type: feature
priority: Medium
status: open

---

## 概要

100日・1年などのマイルストーン達成時に表示されるお祝いカードを PNG 画像として保存・シェアできる機能を Sora+ 限定で実装する。

## 背景

マイルストーン達成はSNSシェア意欲が最も高い瞬間。PNG 保存 → SNS シェアはバイラル獲得の導線になる。また Sora+ の感情的価値を「形に残せる体験」として強化する。

## 目的

マイルストーンを「保存・シェアしたいビジュアル体験」にし、Sora+ 価値の向上と口コミ獲得を同時に実現する。

## スコープ

- `/api/og` に `type=milestone` パラメータを追加し記念日専用デザインを実装
  - 構成: 星空背景・大きな絵文字・記念日ラベル（「100日目」等）・ペット名・「Sora」ロゴ
- マイルストーンオーバーレイに「記念カードを保存する」ボタン追加
  - Plus ユーザー: `/api/og?type=milestone&...` を fetch → Blob URL → `<a download>` でダウンロード
  - Free ユーザー: Upgrade モーダルを表示
- 既存「今日の思い出を残す」「閉じる」ボタンの上に配置

## 非スコープ

- カスタムデザイン選択
- 動画・GIF 出力
- SNS 直接投稿（シェアは既存 `navigator.share` を流用）

## 受け入れ条件

- [ ] Plus ユーザーがカードを PNG でダウンロードできる
- [ ] Free ユーザーには Upgrade モーダルが表示される
- [ ] milestone タイプの OGP 画像が星空デザインで生成される
- [ ] ダウンロードファイル名が `sora-milestone-{label}.png` 形式
