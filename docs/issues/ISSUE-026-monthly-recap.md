# ISSUE-026: 月次ふりかえりカードの自動生成

name: ISSUE-026
type: feature
priority: High
status: open

---

## 概要

月末に「今月の思い出まとめカード」を自動生成し、ホーム画面に表示する。

## 背景

記録が増えても「先月何をしたか」を見返す導線がない。月次サマリーは記録の「意味化」を促し、感情的な満足度（Happiness）を高める。

## 目的

記録が「蓄積→意味→感情価値」の流れになるループを作る。

## スコープ

- 月初（1日）にホーム画面に「先月のふりかえり」カードを表示
  - 表示内容: 記録件数・写真枚数・最多気持ちタグ・先月の先頭写真
- カードタップでその月の記録一覧（カレンダービュー）に遷移
- `lib/recap.ts` に月次集計ロジックを実装（memories 配列から算出、APIコールなし）
- 表示期間: 毎月1〜7日（8日以降は非表示）

## 非スコープ

- 画像書き出し・シェア（ISSUE-027 で分離）
- AI によるコメント生成

## 受け入れ条件

- [ ] 月初1〜7日にふりかえりカードが表示される
- [ ] 件数・写真枚数・気持ちタグが正しく集計される
- [ ] カードタップで先月のタイムラインに遷移する
- [ ] 8日以降はカードが非表示になる
