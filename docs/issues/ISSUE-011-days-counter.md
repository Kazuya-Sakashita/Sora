## 問題の概要

ホーム画面に「今日で〇〇日目」カウンターが存在しない。このカウンターはMVPの中核的な日常利用フックであり、毎日アプリを開く理由になる。

## 詳細

MVP再定義（`docs/specs/sora-product-rethinking.md`）において、「今日で〇〇日目」カウンターはホーム画面のメインコンテンツとして定義された。

**なぜ必要か:**
- 毎日数字が増える → 毎日開く動機になる
- ペットを迎えた初日から計算できる → 新規ユーザーにも即日価値がある
- ロス後も「一緒にいた日数」として永遠に残る → 生前・ロス後をシームレスにつなぐキラー表示
- SNSで「今日でリクと1000日」とシェアしたくなる

## 期待する動作

```
🐾 リクと一緒に
    847 日
```

- ペットプロフィールの「迎えた日（brought_at）」から今日までの日数を計算して表示
- 毎日0時に自動更新（Stateではなく毎レンダリング時に計算）
- 数字を大きく・中央寄せ・温かいフォントで表示
- ペットの名前を上部に添える

## 実装方針

1. `lib/utils/date.ts`（新規）に日数計算ユーティリティを追加
   ```ts
   export function calcDaysWith(broughtAt: string): number {
     const start = new Date(broughtAt)
     const today = new Date()
     const diff = today.getTime() - start.getTime()
     return Math.floor(diff / (1000 * 60 * 60 * 24)) + 1
   }
   ```
2. `home.tsx` にカウンターコンポーネントを追加
3. ISSUE-004（ホーム全面再設計）と合わせて実装する
