## 仕様変更による優先度更新（2026-04-28）

**MVPから Phase 1後半（Month 2〜3）に移動した。**

MVP再定義（`docs/specs/sora-product-rethinking.md`）において、予定管理はMVPの5機能に含めないことを決定した。

理由:
- MVP最初期は「記録習慣の形成」に集中する
- 複雑性を増やす前に、コアの記録体験（ISSUE-001, 011, 012, 013）を完成させる

**着手タイミング:** ISSUE-001, 011, 012, 013 が完了してから着手する
