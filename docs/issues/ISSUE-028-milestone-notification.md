# ISSUE-028: 記念日・マイルストーン通知とお祝いカード

name: ISSUE-028
type: feature
priority: High
status: open

---

## 概要

一緒に過ごした日数の節目（100日・365日・1000日）とペットの誕生日・記念日当日に、アプリ内カードとプッシュ通知で祝う。

## 背景

記念日は感情的に意味のあるタイミング。このタイミングにアプリが「気づいてくれる」体験が、情緒的なロイヤリティを生む。誕生日・記念日はリテンションの最強トリガー。

## 目的

感情的な「特別感」を届け、記念日前後のD1/D7継続率を高める。

## スコープ

- `lib/milestone.ts` に節目日判定ロジックを実装
  - 100日・365日・1000日（一緒に過ごした日数）
  - ペットの誕生日（birthDate）
  - お迎え記念日（broughtAt の年次）
- ホーム画面: 節目当日にお祝いカードをフルスクリーンオーバーレイで表示
- `/api/push/send` のCronロジックを拡張し、記念日当日は特別メッセージを送信
  - 通常: 「今日の思い出を残しませんか」
  - 記念日: 「ポチとの1年記念日です。今日も残しましょう🎂」
- カードにはタイムラインへの導線を設置

## 非スコープ

- カード画像のシェア
- ユーザーカスタム記念日の追加

## 受け入れ条件

- [ ] 100/365/1000日目にお祝いカードが表示される
- [ ] 誕生日・お迎え記念日当日に通知と祝いカードが出る
- [ ] 過ぎた節目は表示されない（当日のみ）
- [ ] カードからタイムラインへ遷移できる
