# ISSUE-021: ホームに「n日前の今日」カードを追加する

## 基本情報

| 項目 | 内容 |
|------|------|
| 優先度 | high |
| ステータス | done |
| 関連画面 | ホーム画面 |
| 関連ファイル | `components/screens/home.tsx` |

## 問題の概要

記録が蓄積しても「発見体験」がない。見返すトリガーがアプリ外に存在せず、蓄積の価値が伝わらない。

## 期待する動作

- 過去同月同日の記録があれば「1年前の今日」「○日前の今日」カードを表示
- 写真があればサムネイル表示
- タップでタイムラインへ遷移
- 過去記録がなければカードは非表示（空振りしない）

## 実装方針

```ts
// 過去同月同日の記録を取得
const today = new Date()
const onThisDay = memories.filter((m) => {
  const d = new Date(m.date)
  return d.getMonth() === today.getMonth() &&
         d.getDate() === today.getDate() &&
         d.getFullYear() !== today.getFullYear()
})
```

- 最新1件を表示（複数あれば最も古いもの）
- 「{n}年前の今日」または「{n}日前の今日」と表示
